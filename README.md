# Flask + PostgreSQL Docker API

A simple containerized backend API built with Flask, PostgreSQL, and Docker Compose.

---

## 🚀 Features

- Flask REST API
- PostgreSQL database integration
- Dockerized multi-container setup
- Environment variable configuration (.env)
- Docker Compose orchestration
- Basic Notes CRUD API

---

## 🛠 Tech Stack

- Python (Flask)
- PostgreSQL
- Docker
- Docker Compose
- psycopg2
- python-dotenv
